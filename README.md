# Animal Encyclopedia Website

This project is a simple web-based Animal Encyclopedia developed using PHP and MySQL.
It allows users to view information about animals through a dynamic website connected to a database.

## Developer
- Modhi Bandar Alshammari

## Technologies Used
- PHP
- HTML
- CSS
- JavaScript
- MySQL
- NetBeans IDE

## How the Project Works
1. The website connects to a MySQL database using PHP.
2. Animal data is stored in the database.
3. PHP files retrieve data from the database and display it on the website.
4. The user can browse animal information through the web pages.

## Database Setup
1. Import the database file (`animal_db.sql`) into phpMyAdmin.
2. Create a database with the same name used in the PHP connection file.
3. Update database connection settings if needed.
